# Patch Notes - 2026-03-31

## Release: v1.5

## Included in this release

- Refined the dark web-shell contrast hierarchy for real-world viewing conditions, especially reflective glossy displays and ambient light.
- Improved ALB download workflow readability with clearer separation between background, passive panels, interactive controls, and stacked right-column sections.
- Added client-side ALB download form validation so missing required inputs are called out before the request is submitted.
- Renamed the ALB action to `Open logs folder`, clarified that `Config name` is optional, and updated the Sentry copy to `Sentry Infrastructure`.
- Polished ALB IP Summary Excel workbook formatting for clearer sectioning and summary readability.

## Web shell contrast

Dark mode now uses a stronger but still restrained hierarchy:

- darker page background
- clearer passive panel surfaces
- more defined interactive surfaces
- stronger input boundaries and focus states
- more readable compact status, progress, and metadata elements

The goal was practical readability under glare and reflections, not a brighter or noisier theme.

## ALB download workflow

The ALB download page received multiple operator-facing usability improvements:

- clearer distinction between passive panels and clickable controls
- better small-button and pill affordance
- stronger field boundaries for inputs/selects/textareas
- clearer right-column panel grouping
- inline warning banner plus red required-field highlighting when mandatory input is missing

## Excel output

The ALB IP Summary workbook formatting was refined to improve scanability:

- stronger section headers
- bordered summary blocks
- clearer metric grouping
- cleaner top-level sheet presentation

## Versioning

- Application version updated to `1.5`
- Assembly version updated to `1.5.0.0`
