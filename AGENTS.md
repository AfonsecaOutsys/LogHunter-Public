# AGENTS

At the end of a coding session:

- Always push the current working branch to GitHub, provided it is not `main`.
- Never merge to `main`.
- Only merge into `main` when explicitly asked to do it.
- After pushing, produce a Release build to `C:\espaces`.
- If `C:\espaces\LogHunter.exe` is in use and cannot be replaced, publish the executable as `C:\espaces\LogHunter-new.exe` instead.
